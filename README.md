# Agent Studio

一个从零开发的 AI Agent 应用项目。

V0.5

新增：

- ES6箭头函数重构
- 普通函数统一改为箭头函数写法
- 匿名函数改为箭头函数写法
- 使用对象解构优化消息数据访问
- 优化历史聊天记录加载逻辑
- 简化消息渲染流程
- 提升代码可读性与现代JavaScript规范
