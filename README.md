# Agent Studio

一个从零开发的 AI Agent 应用项目。


## 当前版本

 V0.4

新增：

- localStorage实现聊天记录持久化
- 页面刷新后自动恢复历史聊天记录
- messages数组与浏览器存储同步
- 聊天记录清空功能
- 空聊天状态提示页面
- 根据聊天状态动态显示/隐藏提示
- 用户消息和AI消息数据结构统一
- 优化聊天数据保存流程
